# TalentScout AI

TalentScout AI is a recruitment platform for PESO that uses a mint-blue visual scheme and AI-assisted hiring workflows.

## Design Scheme

- Primary site color: `#98FBCB` (mint blue)
- Applied across shared navigation, buttons, cards, tags, and module pages

## Core User Roles And Features

### Employees

- Job opportunities browsing and filtering
- Resume builder for profile-ready applications
- Certification/course/upskill suggestions based on skill gaps
- Personal profile management
- Job application tracker for status updates from apply to hire

### Employers

- Job posting management
- Employee finder with blind hiring support
- Job application tracker for screening and hiring stages

### Admin (PESO)

- Central dashboard for platform metrics
- Overview of employers and employees
- Job application tracker and pipeline monitoring

## Platform-Wide Capabilities

- AI document processing and AI-based matching
- Permit or ID registration flow for employers
- Chatting and SMS support for communication and updates

## Current Modules In This Workspace

- `index.html` - Landing page
- `job-postings/` - Job opportunities and posting discovery
- `ai-matching/` - AI match scoring and recommendations
- `skill-gap-analysis/` - Skill analysis and upskill guidance
- `blind-hiring/` - Bias-reduced screening workflows
- `applicant-tracking/` - Application status pipeline
- `peso-dashboard/` - PESO admin overview and monitoring
- `styles/global.css` - Shared site-wide mint-blue theme
